# IoT_Project
IoT project completed during the final year of computer engineering studies.
Purpose: Communication via LoRa antennas between:
- M5Core
- Arduino MKR WAN 1310, connected to an MKR IoT CARRIER REV2 sensor
- Heltec V3, connected to a KY-015 DHT11 sensor 

and publish temperature/humidity/pressure sensor data via MQTT on adafruit.com
